Summary
=======

Python bindings for the `Tokyo Cabinet <http://tokyocabinet.sourceforge.net/index.html>`_ database.

This code is heavily based on `pytc <http://pypi.python.org/pypi/pytc>`_ developed by Tasuku SUENAGA.
